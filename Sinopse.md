# Exercício War (Estrutura de Dados)

## Etapa Novato (Em andamento)
1 - Definir uma struct **Território** com os campos **char nome[30]**, **char cor[10]**, e **int tropas**.
**[FINALIZADO]**

2 - O sistema deve permitir que o usuário **cadastre cinco territórios** informando o nome, cor do exército e o número de tropas de cada um.
**[FINALIZADO]**

3 - O sistema deve **exibir as informações de todos os territórios** registrados após o cadastro.
**[FINALIZADO]**

4 - O código deve conter **comentários explicativos** sobre a criação da struct, entrada e exibição de dados.
***(EM ANDAMENTO)***
